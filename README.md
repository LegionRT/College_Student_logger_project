# College Student Logger Project

Модульная система управления успеваемостью студентов на C++, разработанная в рамках учебной практики.

## 📋 Описание

Система состоит из трёх частей:
- **StudentCore** — ядро с бизнес-логикой обработки данных студентов
- **StudentDLL** — динамическая библиотека, экспортирующая функции ядра
- **StudentApp** — консольное приложение с меню, работающее с DLL через позднее связывание (`LoadLibrary`/`GetProcAddress`)

## 🎯 Возможности

- Создание групп студентов произвольного размера
- Генерация демонстрационных данных (случайные оценки от 2 до 5)
- Расчёт средних баллов
- Поиск лучшего студента
- Подсчёт должников (оценки ниже 3)
- Фильтрация по порогу среднего балла
- Сортировка по убыванию среднего балла
- Безопасная обработка `nullptr`

# College Learning Project

Модульная система управления успеваемостью студентов на C++. Проект разделен на ядро (Static Lib), динамическую библиотеку (DLL), клиентское приложение и модульные тесты.

## 🛠 Требования

- **MinGW / MSYS2** (g++, make)
- **CMake** (нужен только для сборки тестов)
- **Git** (для автоматического скачивания Google Test)

## 🚀 Сборка и запуск приложения

Компиляция основного проекта и запуск консольного меню:

cmd in derectory makefile
- mingw32-make          `Компиляция StudentApp.exe и StudentDLL.dll`
- mingw32-make run      `Запуск приложения`
Сборка и запуск тестов
## Тесты используют Google Test. Перед сборкой нужно один раз подготовить библиотеки
- cmd in derectory Makefile.test
- install_gtest.bat               `Скачивание и сборка Google Test (делается один раз)`
- mingw32-make -f Makefile.test    `Компиляция тестов`
- mingw32-make -f Makefile.test run ` Запуск тестов `
## Очистка проекта
- Удаление скомпилированных файлов (.exe, .dll, .o):
- mingw32-make clean              ` Очистка основного приложения `
- mingw32-make -f Makefile.test clean ` Очистка тестов `
